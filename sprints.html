Relación con Sprints Ágiles

Sprint Actividad Seguridad
1 Login Validación de entrada para evitar datos incorrectos o maliciosos
2 Registro de usuario Verificación de contraseñas seguras y protección de los datos personales del usuario
3 Recuperación de cuenta Validación de identidad para asegurar que solo el usuario legítimo pueda recuperar su acceso
4 Perfil de usuario Autorización para controlar qué información puede ver o modificar cada usuario, y protección de datos personales
5 Pagos o transacciones Cifrado de datos y revisión de seguridad para proteger información sensible como contraseñas o datos bancarios
