Relación con Sprints Ágiles

Sprint Actividad Seguridad
1 Subir avatar de usuario Validación de entrada: se sanitiza el nombre del archivo con getName() para eliminar caracteres peligrosos como "../" o "/".
2 Subir documento PDF Validación de extensión por lista blanca: solo se permiten .jpg, .jpeg, .png, .pdf usando matches().
3 Subir archivo al perfil Doble validación de ruta: se verifica con getCanonicalPath() y startsWith() que el archivo no salga del directorio permitido.
4 Reemplazar archivo existente Control de sobrescritura: se verifica con file.exists() antes de escribir, evitando que se destruyan archivos importantes.
5 Subir archivo grande Límite de tamaño: se controla que el archivo no supere los 5 MB con data.length, evitando denegación de servicio (DoS).
